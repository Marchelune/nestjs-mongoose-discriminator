# nestjs-mongoose-discriminator


A sample app showing how to deal with mongoose's discriminators in [Nest](https://github.com/nestjs/nest).


## Description



## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
